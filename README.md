# WebSocket Echo Service

Simple WebSocket Echo Service for Python.

![demo](./assets/demo.gif)

## Installation

```bash
pip install -r requirements.txt
```

## Server-side

```bash
python server.py
```

## Client-side

```bash
python client.py
```

## Configuration

See `.env`:

```test
HOST=127.0.0.1
PORT=3033
```
